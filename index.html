Projects

Wack-A-Mole

An implemented closed-loop position control system to control and integrate hardware with Simulink and MATLAB to play Wack-A-Mole. Manipulating and reading motor angles and triggering the solenoid through Simulink while communicating between the Quenser board and the MATLAB code. Image processing using a HP webcam and MATLAB to locate and identify RGV values of the LEDs.

Visual Inspection Unit

Autonomous inspection robot capable of identifying holes on a provided surface. Provided user-friendly GUI capable of accessing the file containing the captured holes specification, provide a simulated visual of the surface and captured holes, and start the rover in either “Sweep” or “Inspect” mode. Processed through a Raspberry Pi accompanied by Arduino and Alamode, and image capture through a Raspberry Pi camera module with input connection to the processor.
